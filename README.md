# Movie Website

This application uses the TMDb API but is not endorsed or certified by TMDb.\
`Note: user sign in functionality is disabled in live demo.`

## Tech Stack

- TypeScript
- React
- Recoil
- Nextjs
- Nodejs
- Firebase
- TheMovieDataBase API

[Project Demo](https://movie-website-tan.vercel.app)

## Analytics

This project uses **Vercel Analytics**. The `@vercel/analytics` package is included and the `Analytics` component is mounted in `pages/_app.tsx`. To enable analytics data collection, ensure Vercel Analytics is enabled for the project in the Vercel dashboard.
